## Javascript
```js
 $(document).ready(function(){
	    $("select[name=example]").change(function () {
	    $(".hidden_input").hide();
		var cnt = $("select[name=example]").val();  
			
			while(cnt > 0) {
				$(".hidden_input#"+cnt).show();
				cnt--;	
			}
			
	    });
	 
});
```
